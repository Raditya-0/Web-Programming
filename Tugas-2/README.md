# Tugas 2 - HTML Table & Login Form

Tugas pemrograman web yang menampilkan halaman dengan tabel HTML dan form login sederhana.

## Live Preview

> Buka [Link](https://raditya-0.github.io/Web-Programming/Tugas-2/index.html) di browser untuk melihat tampilan website.


## Deskripsi

Halaman ini terdiri dari dua tampilan utama:
1. **Tabel Nilai** – Menampilkan tabel nilai mata pelajaran Kimia, Fisika, Biologi untuk beberapa siswa.
2. **Form Login** – Form login dengan berisi username, password, dan opsi "Remember me".

## Fitur

- **Navbar** dengan navigasi ke halaman Menu dan tombol Login.
- **Tabel HTML** dengan `rowspan` dan `colspan`, menampilkan nilai siswa:
  - Robby
  - Rendi
  - Alfian
- **Form Login** dengan:
  - Input Username
  - Input Password
  - Checkbox Remember Me
  - Tombol Back (kembali ke tabel)
  - Tombol Login (submit)

## Struktur File

```
Tugas-2/
```

## Cara Menjalankan

1. Buka file `index.html` di browser.
2. Halaman akan menampilkan tabel nilai secara default.
3. Klik tombol **Login** di navbar untuk menampilkan form login.
4. Klik tombol **Back** untuk kembali ke tampilan tabel.
