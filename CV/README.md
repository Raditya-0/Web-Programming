# Personal CV Website

Website ini menampilkan profil, pengalaman, proyek, dan pencapaian.

---

## Live Preview

> Buka [Link](https://raditya-0.github.io/Web-Programming/CV/index.html) di browser untuk melihat tampilan website.

---

## Struktur Proyek

```
CV/
├── index.html              # Halaman utama website CV
├── css/
│   ├── base.css            # Reset & style dasar global
│   ├── navbar.css          # Styling navigasi bar
│   ├── hero.css            # Styling section about
│   ├── timeline.css        # Styling timeline education & experience
│   ├── projects.css        # Styling proyek
│   ├── achievement.css     # Styling section pencapaian
│   ├── footer.css          # Styling footer / contact
│   ├── animations.css      # Keyframe animasi & transisi
│   └── shared.css          # Style umum yang digunakan bersama
└── js/
    ├── main.js             # Entry point 
    ├── particles.js        # Animasi partikel latar belakang (canvas)
    ├── typing.js           # Efek animasi typing teks otomatis
    ├── slideshow.js        # Slideshow foto profil di about section
    ├── observer.js         # Intersection Observer untuk scroll effects
    ├── navbar.js           # Logika navigasi bar 
    ├── projects.js         # Toggle "See More / See Less" project cards
    └── scroll.js           # Scroll progress bar & smooth scroll
```

---

## Sections

1. **About** — Perkenalan singkat dan foto profil
2. **Education** — Riwayat pendidikan 
3. **Experience** 
4. **Projects** 
5. **Upcoming Projects** — Proyek final yang sedang direncanakan
6. **Achievement** 
7. **About Me** — Deskripsi diri dan dream job
8. **Contact** — Email & tautan media sosial 

---

## Bahasa yang Digunakan

- **HTML5** — Struktur dan semantik halaman
- **CSS3** — Styling, animasi, dan layout responsif
- **JavaScript** — Interaktivitas dan logika modular tanpa framework

## Cara Menjalankan

1. Clone atau download repository ini.
2. Buka file `index.html` menggunakan browser (Chrome, Firefox, Edge, dll.).
3. Tidak diperlukan instalasi dependencies atau server khusus.
